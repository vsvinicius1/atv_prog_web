# 🃏 Jogo da Memória

Jogo desenvolvido para a disciplina **GAC116 - Programação Web (2026/1)**

## 🎮 Como Jogar

1. Clique em uma carta para revelá-la
2. Clique em outra carta para tentar encontrar o par
3. Se as duas cartas forem iguais, o par é marcado como encontrado
4. Se forem diferentes, ambas voltam para o verso
5. Encontre todos os 8 pares para vencer!

## 📁 Estrutura do Projeto

```
memoria-game/
├── index.html   → Estrutura da página (HTML)
├── style.css    → Estilização visual (CSS)
├── script.js    → Lógica do jogo (JavaScript)
└── README.md    → Este arquivo
```

## ⚙️ Funcionalidades

- 16 cartas (8 pares de emojis embaralhados aleatoriamente)
- Contador de tentativas
- Timer de tempo
- Efeito de virada 3D nas cartas
- Modal de vitória com resumo
- Botão de reiniciar a qualquer momento

## 🛠 Tecnologias

- **HTML5** — estrutura semântica da página
- **CSS3** — estilização, variáveis CSS, transições e animações
- **JavaScript puro** — manipulação do DOM, eventos e lógica do jogo
